# LANFail

一个轻量、快速、私密的 Android 局域网 SMB 文件访问工具。

LANFail 专注于解决 Android 手机访问电脑共享文件困难的问题，尤其针对部分手机系统自带文件管理功能不完善、局域网文件访问流程复杂等问题。

通过 SMB 协议，LANFail 可以快速访问：

- Windows 共享文件夹
- NAS 存储设备
- Samba 服务器
- 远程 SMB 文件服务器

支持：

- SMB2 / SMB3 协议
- IPv4 网络
- IPv6 网络
- 局域网访问
- 远程服务器访问
- 视频流式传输
- 在线播放局域网视频文件

LANFail 设计理念：

简单、快速、私密、自主可控。

相比部分系统自带的文件互传工具，LANFail 更加轻量，不包含复杂功能、不依赖云服务、不占用大量资源，只专注于文件访问。

它适合：

- 小米等 Android 手机访问 Windows 共享文件
- 家庭电脑文件浏览
- NAS 文件访问
- 局域网视频播放
- 个人服务器管理

LANFail 不需要上传文件到第三方平台，文件始终保存在用户自己的设备中。

小体积、低占用、无广告，让局域网文件访问变得更加简单。
